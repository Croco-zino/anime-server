🎬 Anime Servers API

⭐ REST API | مجاني | بدون مفتاح

ابحث عن الأنمي واستخرج روابط السيرفرات النظيفة (iframe, embed, streamtape, dood...)

🔍 طريقة العمل

✅ الخطوة 1 : أرسل طلب /search?q= للحصول على قائمة الحلقات مع id فريد لكل حلقة.

✅ الخطوة 2 : استخدم /ep?id= (مع الـ id المستلم) لاستخراج جميع سيرفرات المشاهدة النظيفة.

📌 قاعدة API: https://yourdomain.vercel.app (استبدله برابطك بعد الرفع)

🔎 /search GET

يبحث عن أنمي حسب الكلمة المفتاحية ويعرض النتائج مع id الخاص بكل حلقة.

https://your-api.vercel.app/search?q={query}
q (required) – اسم الأنمي بالعربية أو الإنجليزية.

📌 مثال طلب (cURL)

curl "https://your-api.vercel.app/search?q=one%20piece"

📟 مثال JavaScript (fetch)

fetch('https://your-api.vercel.app/search?q=one+piece')
  .then(res => res.json())
  .then(data => {
      console.log(data.results);
      // data.results[0].id  -> الرقم المستخدم للـ /ep
  });
📦 نموذج الرد:
{
  "query": "one piece",
  "count": 12,
  "results": [
    {
      "id": 1,
      "title": "One Piece Episode 1089",
      "image": "https://hondram.site/thumb.jpg"
    },
    ...
  ]
}
🎞️ /ep GET

يستخرج روابط السيرفرات النظيفة (iframe, link, script) بناءً على id الذي حصلت عليه من البحث.

https://your-api.vercel.app/ep?id={id}
id (required) – الرقم التعريفي للحلقة المستلم من /search

🧪 مثال (بعد البحث)

curl "https://your-api.vercel.app/ep?id=1"

🟢 جلب السيرفرات في تطبيقك (JS)

async function getServers(episodeId) {
  const res = await fetch(`https://your-api.vercel.app/ep?id=${episodeId}`);
  const data = await res.json();
  // data.servers → [{ type: "iframe", url: "https://..." }, ...]
  return data.servers;
}
📦 نموذج الرد من /ep 
{
  "id": 1,
  "count": 3,
  "servers": [
    { "type": "iframe", "url": "https://vidmoly.to/e/abc123" },
    { "type": "link", "url": "https://dood.wf/e/xyz789" },
    { "type": "script", "url": "https://streamwish.com/embed/12345" }
  ],
  "source": "https://hondram.site/episode/one-piece-1089?watch=1"
}

💡 ملاحظة: الخاصية type توضح مصدر الرابط (iframe | link | script) ويمكنك استخدامها لتضمين الفيديو مباشرة.

⚠️ ملاحظات هامة للمطورين

  • 🔹 نظام الـ ID : يتم إنشاء الأرقام (1,2,3...) بشكل مؤقت في ذاكرة الخادم. قد تختلف ids من جلسة لأخرى.
  • 🔹 مدة الاستجابة : قد تصل إلى 5-8 ثوانٍ لأن الـ API يجلب البيانات من الموقع المصدر مباشرة.
  • 🔹 السيرفرات المستخرجة : تشمل VidMoly, Dood, StreamWish, StreamTape, وغيرها. يتم فلترة الروابط غير المرغوب فيها تلقائياً.
  • 🔹 نشر الـ API على Vercel : استخدم ملفات app.py + vercel.json + requirements.txt (Flask, requests, beautifulsoup4).
  • 🔹 قيود Vercel : timeout 10 ثوانٍ – قد تحتاج لتحسين الكود أو إعادة المحاولة.
